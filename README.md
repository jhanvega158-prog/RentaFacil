## Descripción
RentaFácil” es una empresa dedicada al alquiler de vehículos que necesita un sistema para 
gestionar su flota. Se te contrata para desarrollar el backend de este sistema con Spring Boot, 
conectado a una base de datos PostgreSQL, junto con una interfaz web sencilla que permita al 
personal de la empresa probar las operaciones del sistema.

## Funcionalidades
- CRUD de vehículos
- Búsqueda por categoría
- Búsqueda por rango de precio
- Búsqueda por modelo
- API REST con Spring Boot
- Base de datos PostgreSQL

## Tecnologías utilizadas
- Java 21
- Spring Boot
- PostgreSQL
- DB
- Postman
- HTML, CSS y JavaScript
